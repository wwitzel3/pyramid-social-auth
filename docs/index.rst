Welcome to Pyramid Social Auth's documentation!
===============================================

Pyramid Social Auth is a pyramid auth plugin that wraps the great `Python Social Auth`_ library
and implements the Velruse_ API. If you currently use Velruse_ this should library should work with
minor modifications. See :ref:`migrating-velruse`.
